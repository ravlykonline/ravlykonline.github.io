# РАВЛИК 🐌

**РАВЛИК** — це перша україномовна текстова мова програмування для дітей з миттєвим візуальним результатом у браузері.

Проєкт допомагає зробити легкий перехід від візуальних блоків до справжнього кодування через малювання фігур, базові алгоритми, власні функції та інтерактивний ігровий режим.

## 🌟 Чому РАВЛИК?

- **Рідна мова:** Усі команди записуються українською мовою.
- **Миттєвий фідбек:** Дитина одразу бачить результат на полотні.
- **Дружній дебаггінг:** Повідомлення про помилки підказують номер рядка та спосіб виправлення.
- **Від простого до складного:** Навчання від руху по прямій до створення ігор з керуванням клавішами.

## 📐 Основні команди та можливості

### Базовий рух та малювання
- `вперед N` / `назад N` — рух на певну кількість кроків.
- `праворуч N` / `ліворуч N` — поворот на кут у градусах.
- `колір [назва]` — зміна кольору олівця (червоний, синій, зелений... та `веселка`).
- `підняти` / `опустити` — керування олівцем для переміщення без сліду.
- `перейти в X Y` — миттєве переміщення у координати (X, Y) від центру.
- `очистити` — видалення малюнка з полотна.

### Логіка та управління
- `повторити N ( ... )` — цикл для малювання візерунків та фігур.
- `якщо умова ( ... )` / `... інакше ( ... )` — прийняття рішень у коді.
- `якщо край ( ... )` — перевірка зіткнення з межею полотна.

### Просунуті можливості (V4+)
- **Змінні:** `створити назва = число` та подальше оновлення `назва = нове_число`.
- **Власні команди (Функції):** `створити назва(параметр) ( команди )`.
- **Ігровий режим:** `грати ( ... )` — безперервний цикл для реакції на клавіші `"вгору"`, `"вниз"`, `"ліво"`, `"право"`.

## 🛠️ Як почати

1. Відкрийте [онлайн-редактор](https://ravlyk.org/).
2. Введіть код для малювання квадрата:
```ravlyk
колір червоний
повторити 4 (
  вперед 100
  праворуч 90
)
```
3. Натисніть **🚀 Запустити**.
