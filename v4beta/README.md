# Равлик

**Равлик** — це проста україномовна мова програмування для дітей з миттєвим візуальним результатом у браузері.

Проєкт допомагає зробити перші кроки в текстовому програмуванні через малювання фігур, базові алгоритми, цикли, умови, змінні та простий інтерактивний режим.

## Чому Равлик

- Команди записуються українською мовою.
- Дитина одразу бачить результат на полотні.
- Помилки показуються у дружньому форматі.
- Навчання йде від простих команд до циклів, умов і режиму `грати (...)`.

## Спробувати онлайн

- Онлайн-редактор: <https://ravlyk.org/index.html>
- Посібник: <https://ravlyk.org/manual.html>
- Уроки: <https://ravlyk.org/lessons.html>

## Основні команди

- `вперед N`
- `назад N`
- `праворуч N`
- `ліворуч N`
- `колір назва`
- `підняти`
- `опустити`
- `очистити`
- `повторити N ( ... )`
- `якщо умова ( ... )`
- `грати ( ... )`

Повний опис мови і приклади використання дивіться у посібнику: <https://ravlyk.org/manual.html>

## Як почати

1. Відкрийте онлайн-редактор.
2. Введіть кілька простих команд, наприклад:

```ravlyk
колір синій
вперед 100
праворуч 90
вперед 50
```

3. Натисніть `Запустити`.
4. Подивіться результат на полотні.
