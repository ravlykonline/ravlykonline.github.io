Довідка: Команди Равлика

  • вперед N — рух вперед на N кроків
  • назад N — рух назад на N кроків
  • ліворуч N — повернути ліворуч на N градусів
  • праворуч N — повернути праворуч на N градусів
  • колір [назва] — змінити колір (напр. червоний, синій, веселка)
  • перейти в X Y — перейти у точку (X, Y) від центру полотна
  • підняти — підняти олівець (не малювати при русі)
  • опустити — опустити олівець (малювати при русі)
  • очистити — очистити тільки малюнок (код не змінюється)
  • повторити N ( ... ) — повторити команди в дужках N разів
  • створити назва(параметр) ( ... ) — створити власну команду
  • назва(число) — викликати власну команду з числом
  • створити змінна = число — створити змінну
  • змінна = число — змінити значення змінної
