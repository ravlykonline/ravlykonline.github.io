Правила

Вікторина містить 10 питань. Обери 1 варіант у кожному питанні й натисни “Перевірити результат”.

1) Яка команда рухає Равлика вперед?
2) Яка команда змінює колір олівця?
3) Для чого потрібна команда повторити?
4) Що робить команда очистити?
5) Яка пара команд відповідає за поворот?
6) Що робить умова якщо край ( ... )?
7) Для чого команда підняти?
8) Що означає запис створити n = 5?
9) Для чого блок грати ( ... )?
10) Що корисно зробити, якщо програма не працює?
